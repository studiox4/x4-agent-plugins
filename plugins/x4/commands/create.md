Run the `/create` skill to scaffold a new x4-mono project.

Uses `bunx create-x4` under the hood with interactive configuration for
presets (full-stack, saas, landing, api-only), platform selection,
npm scope, and environment setup.

Requires Bun >= 1.1 and Git.
